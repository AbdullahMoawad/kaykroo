# kaykroo

### About the application
    This is a demo to allow you create list and update user app using RESTful API .
    
### How to setup
    * docker-compose up
    
    * backend
    
        - yarn
        - yarn dev
        
    * frontend
    
        - yarn
        - yarn dev   

### Used Tools
    * typescript
    * expressjs
    * nextjs
    * typeorm
    * axios
    * formik
    * nodemon

### What more can be done

    * Listing filters and sorting
    * Proper user authentication
